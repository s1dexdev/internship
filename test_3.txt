1. Что такое функция высшего порядка? Аргументированный пример использования,
там где действительно необходимо?

1) Функция высшего порядка это функция которая принимает в качестве аргумента
другую функцию или возвращает другую функцию как результат своей работы. Например:
- функции которые принимают callback;
- замыкание;
- мемоизация.

2. Что такое ассинхронный итератор и генератор? Принцип его работы и применение
с помощью for await of.

2) Ассинхронный итератор и генератор позволяют итерировать данные которые приходят
ассинхронно. Со стека попадают в Web API браузера затем в stack queue.
У итератора есть метод next который возвращает объект со свойствами {value: .../undefined и done: true/false};
Генератор возвращает объект итератора и позволяет выполнить код в функции.
