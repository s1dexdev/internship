2. Синтаксис наследования классов и как в этом участвует ключевое слово super?

2) Метод super() вызывает конструктор родительского класса и позволяет
использовать свойства родительского класса внутри дочернего. Метод super 
необходимо вызывать первым в конструкторе дочернего класса. Синтаксис:

class Figure {}
	
class Rectangle extends Figure {
  constructor() {	
    super();	  
  }
}
